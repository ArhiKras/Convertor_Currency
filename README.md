# Конвертер Валют

Программа для конвертации валют с использованием актуальных курсов обмена.

## Возможности

- **Конвертация валют**: Перевод суммы из одной валюты в другую
- **Просмотр курсов**: Показ текущих курсов обмена между валютами
- **Список валют**: Просмотр всех доступных валют (166 валют)
- **Обновление данных**: Синхронизация курсов с API exchangerate-api.com
- **Поиск валют**: Поиск валют по коду

## Структура проекта

```
Convertor_Currency/
├── currency.py          # Точка входа в приложение
├── api_client.py        # Модуль для HTTP-запросов
├── storage.py           # Модуль для управления данными и I/O
├── cli.py               # Модуль интерфейса командной строки
├── currency.json        # Кэшированные данные о курсах валют
├── requirements.txt     # Зависимости проекта
├── README.md            # Документация
└── .gitignore           # Игнорируемые файлы Git
```

### Описание модулей

- **`currency.py`** - Точка входа в приложение, запускает CLI интерфейс
- **`api_client.py`** - Работа с API exchangerate-api.com для получения курсов валют
- **`storage.py`** - Управление хранением данных, кэшированием и операциями с файлами
- **`cli.py`** - Интерфейс командной строки с меню и валидацией пользовательского ввода

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

1. Запустите программу: `python currency.py`
2. Программа автоматически проверит актуальность данных:
   - Если файл `currency.json` существует и моложе 24 часов - данные загрузятся из файла
   - Если файл отсутствует или старше 24 часов - данные обновятся из интернета
3. Выберите действие из меню (1-5)
4. Следуйте инструкциям на экране

## Доступные валюты

Программа поддерживает 166 различных валют, включая:
- USD (Доллар США)
- EUR (Евро)
- GBP (Британский фунт)
- RUB (Российский рубль)
- И многие другие...

## Особенности

- **Автоматическое управление данными**: Программа автоматически проверяет актуальность кэшированных данных
- **Кэширование**: Данные хранятся локально в файле `currency.json` для быстрого доступа
- **Автообновление**: Данные обновляются автоматически, если файл старше 24 часов

## Источник данных

Курс валют обновляются с помощью API exchangerate-api.com.
Для принудительного обновления данных используйте опцию 4 в главном меню.

## Системные требования

- Python 3.6+
- Интернет-соединение для обновления курсов
- Установленные зависимости (requests, colorama)
